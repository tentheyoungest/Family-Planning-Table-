Mga Bahagi ng Reproductive System ng Lalaki

Talakayin:

1. Mga bahagi ng Reproductive System ng Lalaki at ang gamit ng mga ito:
Bahagi Gamit
Testes Ang gumagawa ng semilya at ng panlalaking hormon na nagbibigay ng mga katangiang panlalaki.
Epididymis Dito naiipon at nagma-mature ang punlay.
Anurang Punlay (Vas Deferens) Ang daanan ng semilya mula sa testes patungong anurang ihi (urethra) at tuloy sa titi (penis).
Titi Ang pangunahing bahagi ng Reproductive System ng Lalaki.
2. Ejaculation

Ito ay ang paglabas ng Tamod na nagtataglay ng punlay at ang kahalagahan nito sa pagbuo ng bata.

Ang kalidad at dami ng punlay ay isang batayan ng kakayahan ng Lalaki na makabuo ng bata.

Mga Bahagi ng Reproductive System ng Babae

Talakayin:

Mga Bahagi ng Reproductive System ng Babae at ang gamit ng mga ito:
Bahagi Gamit
Obaryo (Ovary) Gumagawa ng itlog at ng pambabaeng hormon na nagbibigay ng mga katangian ng Babae.
Anurang Itlog (Fallopian Tube) Daanan ng hinog na itlog mula sa obaryo patungo sa bahay bata.
Matris (Uterus) Bahay bata o ang binabahayan ng nabubuong bata.
Liig-liigan (Cervical Canal) Ang daan patungo sa bahay bata.
